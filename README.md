# image-to-pdf-tool
图片转 PDF 高效处理工具 (Image to PDF Tool)



一款基于 Python 和 Tkinter 开发的现代化桌面工具，支持将多种图片格式快速、无损地转换为 PDF 文件。

## ✨ 功能特点
* **经典布局**：左侧源文件，右侧输出结果，滚轮强制同步对齐。
* **原生体验**：支持双击列表项直接唤起 Windows 默认程序进行预览。

## 🚀 快速运行
* **绿色软件**：windows下载exe即可使用
